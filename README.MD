# swagger-ui

## 预览

  [https://swag.iosoon.cn](https://swag.iosoon.cn)

## 优点

- 操作方便🐶
- 可以同时部署多个项目的api文档🚀

## 快速使用

- 根据[模版配置](./nginx.conf)配置nginx
- 把swagger生成的接口文件命名为项目名+环境名+json。如`project.env.json`
- 把`project.env.json`放到docs目录下
- 执行`./swag-ui`即可

## 自定义

- 修改main文件
- 重新编译swag-ui命令

  ```
  go build -ldflags "-s -w" -trimpath -o swag-ui .
  ```

## 开发
  - 初始化
  ```bash
  go run . init
  ```
  - 运行
  ```bash
  go run . serve
  ```
